# 11653: : 소인수분해

소인수 분해

n = p * q

예를 들면 18을 소인수 분해한다.

18 / 2 = 9, 9 / 3 = 3, 3/3 = 1

18 = 2*3*3

for(int i = 2; i ≤ root(n); i++)

1)나눠지면 (n % i == 0) → n = n / i;

cout >> n;

```bash
#include <iostream>
using namespace std;

int main()
{
    int N;
    cin >> N;
    for(int i = 2; i * i <= N; i++)
    {
        while(N % i == 0)
        {
            cout << i << '\n';
            N /= i;
        }
    }
    if(N > 1)
        cout << N << '\n';

    return 0;
}

```

    - 바깥 루프 (for)
