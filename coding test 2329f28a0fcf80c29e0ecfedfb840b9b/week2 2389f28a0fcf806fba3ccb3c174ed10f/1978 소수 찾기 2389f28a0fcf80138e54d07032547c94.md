# 1978 : 소수 찾기

1. 소수 판정
    
    소수는 1과 자기 자신으로만 나눠지는 수
    
    판정 → 2부터 반복문 돌면서 if-else 조건문으로 소수 판정
    
    - 소수가 아닐 경우(n % i == 0) → break
    - 소수 일 경우 (n % i ≠ 0) → cnt++
2. 시간복잡도 : O(n^2)
3. 코드

```bash
#include <iostream>
using namespace std;

int main()
{
    int N, num;
    int count=0;
    cin >> N;

    int i;
    while(N--)
    {
        cin >> num;
        for(i = 2; i < num; i++)
        {
            if(num % i == 0) break;

        }
        if (i==num) count++;
    }
    cout << count;
    return 0;
}
```
